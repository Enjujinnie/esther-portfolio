# Esther Njuguna — Portfolio

A personal portfolio website showcasing 16 projects across web development, AI/ML, cloud, and design.

**Live site:** https://estherenjuguna.netlify.app (update after deploy)

## Stack

Pure HTML, CSS, and JavaScript — no build tools, no frameworks. Single-file architecture with an interactive project modal system, category filters, and smooth scroll animations.

## Features

- Fully responsive dark navy + gold design
- 16 featured projects with detailed modals
- Category filtering (Web Dev, Full-Stack, AI/ML, Design, Cloud)
- Functional contact methods (email, phone, WhatsApp)
- Smooth scroll and fade-in animations
